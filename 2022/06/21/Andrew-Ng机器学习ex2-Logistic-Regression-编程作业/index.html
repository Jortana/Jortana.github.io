2 正则化逻辑回归 Regularized logistic regression

在这一部分中，我们需要用到 正则化逻辑回归 Regularized logistic regression ，现在有一批芯片测试的结果，每一个芯片都进行了两轮的测试，并且标记出了合格与不合格，现在我们要根据这两个测试的结果判断这个芯片是否合格。

在这个任务中我们需要完成：

  • 数据可视化
  • 特征映射
  • 计算 cost function 和 gradient
  • 用 fminunc 优化计算
  • 可视化决定边界
  • 【可选】调整参数
